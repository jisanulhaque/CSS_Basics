CSS Integration

CSS can be integrated with html in three ways:

  • Inline CSS: Written in the same line with html tag/elements.
  • Internal CSS: Written in the "style" element.
  • External CSS: Written in a seperate file of css which is integrated with html page.

CSS Power

So as mentioned above CSS can be integrated in 3 ways. So which one will be executed if three off these used on the same element?

Inline/Internal/External ??

Its definitely the Internal though External is always the standard!

CSS Selector

CSS has three types of selectors:

  • Tag/Elements Selector
  • CLass Selector
  • ID Selector

CSS Syntex

Selector {

property: value ;

}

So CSS selector will be executed in the following sequence:

ID>Class>Tag

CSS Colors

CSS has four types of colors:

  • Solid color: Red, Blur etc.
  • Hexadecimal: #000000
  • RGB: rgb(11, 11, 11)
  • RGBA: rgba (11, 11, 11, 1)

Here in RGBA the last digit represents the transperency.
